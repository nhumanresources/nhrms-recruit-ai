⚙️ Configure Google Sign-In

You need a free Google Client ID to enable Sign in with Google. Takes ~2 minutes:

  1. Go to console.cloud.google.com
  2. Create a project → APIs & Services → Credentials
  3. Click Create Credentials → OAuth Client ID
  4. Select Web application
  5. Under Authorised JavaScript origins, add:
    https://nhumanresources.github.io
  6. Copy the Client ID and paste below
